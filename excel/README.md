# Excel Analysis & Visual Preparation

This folder contains the Excel workbook used for data exploration, aggregation, and visualization preparation based on the cleaned layoffs dataset.

## What Was Done in Excel
- Imported cleaned data exported from SQL
- Created pivot tables to summarize:
  - Layoffs by company
  - Layoffs by industry
  - Layoffs by country
  - Layoffs by funding stage
- Grouped layoffs by year and month
- Prepared aggregated tables for chart creation

## Purpose of Excel in This Project
- Acts as a bridge between SQL analysis and visual storytelling
- Used for quick aggregation and validation of SQL results
- Supports static charts and future dashboard creation

## Notes
- Core data cleaning and EDA were performed in SQL
- Excel was used mainly for summarization and visualization support
