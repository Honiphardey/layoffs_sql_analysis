# Visualizations Overview

This folder contains visual representations derived from the SQL and Excel analysis of the layoffs dataset.

## Visuals Included
- Layoffs trends over time (yearly and monthly)
- Top companies by total layoffs
- Top industries by total layoffs
- Top countries by total layoffs
- Layoffs distribution by funding stage
- Top locations by total layoffs

## Visualization Goals
- Present insights in a clear and intuitive way
- Support findings from SQL exploratory analysis
- Create shareable visuals for documentation and social media

## Notes
- Visuals are static images generated from Excel
- Interactive dashboards may be added in future updates
